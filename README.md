# README
## Purpose:
This Capture the Flag challenge is meant to test basic Linux/shell commands

## Organization:
Each level will have a hint as to what the flag is (which letter/character).  
Retrieving all of the flags from the 5 levels will give you the letters/characters (in the order of the levels) to form the final phrase.  

## How to run:
Start the challenge by first running ./setup_ctf.sh  
Then, begin!
